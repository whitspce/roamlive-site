Alpha

Roam is in early alpha. The core stream-from-phone path works, but expect bugs, missing features, and rough edges.

Feedback is the most useful thing you can give right now. Send anything to feedback@roamlive.app: bug reports, feature requests, what’s annoying, what’s missing, all of it. Or open a GitHub issue if you’d rather make bug reports public.

Features

  • Lightweight by design

    Roam is small, focused, and doesn’t pile on background services like Google Play, analytics, SDKs you didn’t ask for, etc.

  • Stream to anywhere

    Kick, Twitch, Youtube, Trovo, your own server, it doesn’t matter which platform you use, this is built on the standard protocol so just Paste the full RTMP URL and hold to go live.

  • Stealth mode

    One tap and the screen goes black but the stream keeps going. Press and hold anywhere to bring it back. Useful for situations where you don’t want to draw attention to yourself (Like being IRL in sketchy areas) or look like an obnoxious streamer (Such as at concerts/events where it’s dark around you). Press and hold anywhere to bring the screen back.

  • BRB screen

    One tap BRB screen with custom text built in. Mutes camera and mic with one press.

  • Auto-reconnect

    If you lose signal Roam will notice and attempt to reconnect until you get signal again (Or until it times out).

  • Predictive thermal management

    The phone tells you before it overheats and drops your bitrate to keep the stream alive. Surfaces what it’s doing instead of silently throttling. Option to turn off if you don’t want any throttling.

  • Chat panel, for you only

    Read Twitch and Kick chat without it burning it into your broadcast.

Upcoming features

Private by design

  • Google Play services not required to run
  • No telemetry, no analytics, no tracking
  • Talks only to the streaming server you point it at
  • Open source on GitHub underGPL‑3.0

How to install

  1. Tap the download button above. You’ll get an .apk file to download.
  2. Open the .apk file. Android will warn you that it’s not from the Play Store (which is by design). When it asks, allow your file manager (or whatever app you opened it from) to install unknown apps.
  3. Open Roam Live app and grant it camera + mic permissions. Go to the Roam Live settings and enter your stream URL + Key. Lastly, enter your username and enable chat if you want it.
  4. Press and hold to go live
